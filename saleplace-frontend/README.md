# 🛒 SalePlace

Full-stack marketplace platform built with NestJS and Next.js using GraphQL.

Users can create, manage, and browse products, communicate in real-time, and securely purchase items.

---

## 🚀 Features

- 🔐 Authentication & user accounts
- 📦 Product creation & management
- 🖼 Image upload (external API)
- 🔎 Search & advanced filtering
- 📄 Pagination
- 💬 Real-time chat (GraphQL subscriptions)
- 💳 Stripe payments integration
- 🗑 Soft delete system
- ⚡ Event-driven architecture (EventBus, Pub/Sub)

---

## 🛠 Tech Stack

### Backend
- NestJS
- GraphQL
- Prisma ORM
- PostgreSQL
- Redis
- BullMQ (queues)
- EventBus & Pub/Sub
- Webhooks
- Stripe

### Frontend
- Next.js
- Apollo Client
- GraphQL
- TypeScript
- TailwindCSS / shadcn UI
